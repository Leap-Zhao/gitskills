# sublime快捷键

## 1 mac下默认快捷键

1. 将屏分为n列: option + command + n

2. 将一个文件切换到第n列屏上: control + shift + n

3. 切换到第n列: ctrl + n 

4. 在一个屏中切换到第n个窗口: command + n 

5. 在一个屏中相互切换: control + tab

6. 新建一个文件: command + n

7. 关闭当前文件: command + w

8. 保存当前文件: command + s

9. 查找一个文件 command + shift + f

10. 回到行首 command + 左方向键
    回到行尾 command + 右方向键
    回到文件头 command + 上方向
    回到文件尾 command + 下方向
